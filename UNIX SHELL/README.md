# Kash — Kassim's Shell

A Unix shell built in C from scratch.

## Features (in progress)
- [x] Basic REPL loop
- [x] Command parsing
- [ ] Command execution (fork/exec)
- [ ] Built-in commands (cd, exit)
- [ ] Pipes
- [ ] Redirection

## Build
```bash
gcc main.c -o kash
./kash
```
